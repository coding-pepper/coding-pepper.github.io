defined standards

functions

stack means that you push arguments on the stack in reverse order
so myfn(a, b, c) you push c then b then a


writef(file_desc=rdi, string=rsi, var_args=stack); bytes_written=rax

formats:
  %i32% int 32bit
  %i16% int 16bit
  %i8% int 8bit
  %i64% int 64bit
  %i128% int 128bit
  %ui32% uint 32bit
  %ui16% uint 16bit
  %ui8% uint 8bit
  %ui64% uint 64bit
  %ui128% uint 128bit
  %s% null-terminated string
  %c% character 8bit
  %f% float 32bit
  %p% pointer in hex (pointer sized)
